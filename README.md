# dnd-project

## Objetivo
Criar uma página onde se possa escolher raças, classes e as diversas features de um personagem de dnd da 5a edição e que seja possível criar uma ficha do personagem.

## Planejamento
() Criar Estrutura
    () Seção 1 - Builder
        () Initial Info
            () Name
            () Race
            () Class
        () Attributes
        () Skills and Attributes Saves
        () Features
            () Feats
            () Racial Features
            () Class Features
        () Proficiencies
        () Being
            () Trait
            () Ideal
            () Bond
            () Flaw
    () Seção 2 - Levelling
    () Seção 3 - Spells
    () Seção 4 - Ficha
